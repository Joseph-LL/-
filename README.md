# -一个简单但功能强大的文件传输工具，支持多种文件格式的批量传输、移动和复制操作。

## 功能特点

- 支持多种文件格式筛选
- 保持目录结构选项
- 文件覆盖控制
- 实时进度显示
- 详细的操作日志
- 现代化深色界面
- 智能文件重名处理

## 支持的文件格式

- 图片: .jpg, .jpeg, .png, .gif, .bmp
- 文档: .txt, .doc, .docx, .pdf, .ppt, .pptx, .xls, .xlsx
- 压缩文件: .zip, .rar, .7z
- 视频: .mp4
- 音频: .mp3
- ## 使用说明

1. 选择源文件夹（包含要传输的文件的文件夹）
2. 选择目标文件夹（文件将被传输到此处）
3. 选择要传输的文件类型
4. 根据需要设置以下选项：
   - 保持目录结构
   - 覆盖已存在文件
   - 移动/复制文件
5. 点击"开始传输"按钮
